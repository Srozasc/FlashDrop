# Task: Mejoras Mobile FlashDrop 2025

Este documento detalla el plan de acción para transformar la experiencia móvil de FlashDrop hacia un estándar premium y moderno.

## ✅ Bloque 1: Descubrimiento y Home (Completado)
- [x] **Hero Discovery Screen**: Crear una pantalla de inicio impactante con:
    - [x] Buscador inteligente prominente.
    - [x] Categorías con iconos minimalistas.
    - [x] Sección "Cerca de ti" con geolocalización.
- [x] **Filtros Inteligentes 2025**: Implementar barra de filtros rápidos:
    - [x] Tiempo de entrega estimado.
    - [x] Envío gratis ($0).
    - [x] Etiquetas de dieta (Vegano, Sin TACC, etc.).
- [x] **Story-mode para Comercios**: Implementar carrusel de historias tipo Instagram para ofertas del día.

## ✅ Bloque 2: Experiencia de Pedido (Completado)
- [x] **Skeleton Loaders Premium**: Reemplazar spinners por esqueletos animados.
- [x] **Smart Cart Modal**: Implementar *Bottom Sheet* para gestión de carrito rápida.
- [x] **One-Tap Checkout**: Integrar pagos rápidos y guardado de tarjetas.
- [x] **Upselling Sugerido**: Algoritmo de "Otros usuarios también compraron...".

## ✅ Bloque 3: Seguimiento y Mapas (Completado)
- [x] **Live Activities**: Notificaciones dinámicas en pantalla de bloqueo (iOS/Android).
- [x] **Timeline Interactivo**: Seguimiento enriquecido (Cocinando, Empacando, En camino).
- [x] **Mapa 2.0**: Seguimiento con polilíneas suavizadas y puntos de referencia.

## ✅ Bloque 4: Fidelización y UI (En progreso)
- [x] **Gamificación**: Sistema de niveles de usuario (Bronce, Plata, Oro).
- [x] **Gestión de Direcciones**: Alias e iconos personalizados para ubicaciones guardadas.
- [x] **Dark Mode Nativo**: Soporte completo para tema oscuro OLED.


## ✅ Bloque 5: Optimización Vercel-React (Finalizado)
- [x] **Carga Progresiva de Imágenes**: Implementar BlurImage con efecto fade.
- [x] **Deduplicación de Datos**: Integrar React Query para navegación instantánea.
- [x] **Ajuste de Padding Global**: Corrección de solapamiento de contenido con la barra de navegación flotante.

## ✅ Bloque 6: Eficiencia y Logística Repartidor (Bloque A)
- [x] **Mapas de Calor (Hotspots)**: Visualización en tiempo real de zonas con alta demanda de pedidos.
- [x] **Integración de Navegación In-App**: Botones de acceso rápido para abrir direcciones en Waze o Google Maps.
- [x] **Prueba de Entrega Digital**: Sistema de confirmación mediante fotografía geolocalizada o PIN único del cliente.

## ✅ Bloque 7: Finanzas y Retención Repartidor (Bloque B)
- [x] **FlashDrop Wallet**: Desglose financiero completo (Base, Propinas, Bonos) con historial de transacciones.
- [x] **Retiro Instantáneo (Cash Out)**: Funcionalidad para que el repartidor transfiera sus ganancias de forma inmediata.
- [x] **Driver Pro Levels**: Sistema de fidelización para repartidores con beneficios y tarifas preferenciales por nivel.

## ✅ Bloque 8: Seguridad y Soporte Repartidor (Bloque C)
- [x] **Centro de Seguridad (SOS)**: Botón de pánico 24/7 y compartición de ruta en tiempo real.
- [x] **Soporte Prioritario VIP**: Integración visual de línea directa para repartidores.
- [x] **Reporte de Incidentes**: Formulario rápido para notificar accidentes o averías.
